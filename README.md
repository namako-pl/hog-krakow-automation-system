# HOG Kraków - Automated Event Management System

## Project Overview
A custom-built digital ecosystem designed to automate content management for the HOG Kraków platform. The system focuses on eliminating manual updates by bridging the frontend with the Google Cloud ecosystem.

## Key Features
- **Google API Integration:** Full synchronization with Google Calendar for real-time management of events (wydarzeń).
- **Automated Media Handling:** Direct integration with Google Drive for automated photo gallery updates.
- **Security-First Architecture:** Implemented secure API handshake protocols and data validation to ensure system integrity.
- **Zero-CMS Cost:** Developed a bespoke backend in PHP, removing the need for expensive third-party CMS subscriptions.

## Tech Stack
- **Backend:** PHP, Google Cloud Console (APIs).
- **Frontend:** HTML5, CSS3, JavaScript.
- **Security:** Data encryption, API Key management.

## Results
Achieved 100% automation in event scheduling and media deployment, significantly reducing operational overhead.
